Glossary
========

.. glossary::

    array-like
        An object that can be interpreted as an array, including :class:`numpy.ndarray` and (nested) sequences. Any object accepted by :func:`numpy.array` is array-like.

    field array
        Any subclass of :class:`numpy.ndarray` and :class:`cicada.arithmetic.Field` that stores integers and implements the semantics of a Galois field.
